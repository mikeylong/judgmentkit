# JudgmentKit 0.6.5

Released: 2026-07-02

JudgmentKit 0.6.5 is a patch release that makes internal design-system acceptance a hard generation constraint instead of advisory review feedback.

## Fixes

- Fails UI implementation candidates that omit design-system provenance, provide incomplete provenance, or claim a source that does not match the accepted internal design system.
- Marks failed design-system candidates as diagnostic-only artifacts so release matrices and public examples cannot treat them as accepted UI evidence.
- Updates MCP review cards, generation handoffs, and repair instructions to send agents back through JudgmentKit with the required design-system constraints before acceptance.
- Regenerates model UI examples and release fixtures so accepted artifacts carry design-system provenance while failed attempts remain diagnostic evidence.
