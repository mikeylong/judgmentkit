# JudgmentKit 0.6.0

Released: 2026-06-21

JudgmentKit 0.6.0 makes design-system provenance a contract authority for generated UI work.

## Improvements

- Added `implementation_contract.design_system_source` with explicit `judgmentkit_default` and `external_design_system` modes.
- Added complete external design-system adapter support for tokens, fonts, icons, and components.
- Added fail-fast validation for incomplete external design-system authority.
- Added a hard `design_system_provenance` implementation review gate.
- Updated MCP structured content, docs, examples, and model UI fixtures to carry the active design-system source.

## Compatibility

- JudgmentKit remains the default design-system authority when no external adapter is supplied.
- Declared external design systems own visual tokens, typography, icon assets, and renderer/component authority.
- External design-system mode does not implicitly fall back to JudgmentKit assets.
