# JudgmentKit 0.6.3

Released: 2026-07-01

JudgmentKit 0.6.3 is a patch release for surface-type recommendation repairs from the product-designer smoke run.

## Fixes

- Adds purpose evidence to surface-type recommendations so designers can see which activity signals drove the fit.
- Improves close-call routing for dashboard monitors, setup/debug tools, marketing lead forms, and AI-assisted review workflows.
- Adds paired surface-type regressions that cover the smoke-run misses and guard against overcorrection.
- Documents prompting patterns that steer real-world briefs toward the intended surface type without exposing implementation machinery.
