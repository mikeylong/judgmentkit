# JudgmentKit 0.3.0

Released: 2026-06-15

## First Shippable AI-Native Contract

JudgmentKit 0.3.0 packages the AI-native design system work as a shippable contract-and-repair loop.

- Bumps package, activity contract, and MCP metadata to `0.3.0`.
- Keeps the public contract surface stable: `default_ai_native_design_system`, `iteration_policy`, `next_agent_action`, `autofix_loop`, `repair_instructions`, and boundary-only `visual_token_adapter`.
- Adds replayable first-use fixtures under `examples/ai-native-design-system/`.
- Adds canonical setup/onboarding, operational dashboard, and high-stakes review examples.
- Adds package install/import smoke coverage from an `npm pack` artifact.

## Renderer Boundary

The renderer/component package remains deferred.

- `visual_token_adapter` stays semantic metadata only.
- Token evidence cannot satisfy missing primitives, states, action boundaries, data visibility, accessibility, static checks, or browser QA.
- No A2UI, catalog/compiler, renderer package, component package, or visual-quality scoring is introduced in this release.
