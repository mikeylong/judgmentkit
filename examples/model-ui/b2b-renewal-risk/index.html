A customer success manager reviews renewal risk and decides save plan, executive escalation, or wait for usage evidence. Three generation paths are shown across four context boundaries. JudgmentKit skill context improves activity fit, workflow fit, disclosure discipline, and implementation guidance; Material UI improves visual/component consistency.
