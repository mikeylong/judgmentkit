Field service dispatch model UI generation matrix

A field operations manager assigns, reschedules, or escalates a repair visit using route, parts, and SLA constraints. Three generation paths are shown across four context boundaries. JudgmentKit skill context improves activity fit, workflow fit, disclosure discipline, and implementation guidance; Material UI improves visual/component consistency.
