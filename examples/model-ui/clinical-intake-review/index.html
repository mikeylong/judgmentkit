Clinical intake review model UI generation matrix

An intake coordinator reviews administrative appointment readiness and decides schedule, return missing forms, or escalate insurance verification. Three generation paths are shown across four context boundaries. Material UI improves visual/component consistency; JudgmentKit improves activity fit, workflow fit, and disclosure discipline.
