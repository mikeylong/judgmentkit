JudgmentKit MCP Private Pilot

Private repo-local pilot. Scores are deterministic checks against saved model captures; JudgmentKit is not used as the judge. Default mode scores existing versioned captures only. Missing or MCP-version-mismatched captures are reported as capture-required. Live capture is opt-in with --capture and reuses existing captures in the target capture directory. Repair-loop results are deterministic fixture proof of contract governance and agent-owned resubmission behavior; they do not measure visual quality or live-model repair quality. Repair-loop observation is an opt-in Codex CLI dogfood pass of agent-owned repair behavior; it is not a deterministic fixture proof, visual-quality score, or pass/fail gate.
